# ✦ RUNEFALL

**呪文でダンジョンを切り拓け**

ルーンタイルを組み合わせて呪文を発動し、8フロアのダンジョンを攻略するスマートフォン向けローグライク×パズルゲームです。

🎮 **[今すぐプレイ](https://ki-programming-project.github.io/runefall/)**

---

## ゲーム概要

| 項目 | 内容 |
|------|------|
| ジャンル | ローグライク × パズル |
| プラットフォーム | スマートフォンブラウザ（PC でも動作） |
| プレイ時間 | 1ラン 15〜20分 |
| 難易度 | EASY / NORMAL / HARD の3段階 |

---

## 特徴

- **タイル組み合わせで呪文を発動** — 6属性のルーンタイルを最大3枚選択。11種類のコンボ呪文が存在し、組み合わせを覚えるほど有利になります
- **毎回異なるダンジョン** — フロア構成はランダム生成。戦闘・ショップ・休憩・宝箱・イベントが混在します
- **3段階の難易度** — EASYは初心者向けの手厚いサポート。HARDはデッキ上限20枚制限でデッキ圧縮戦略が求められます
- **スコアシステム** — 到達フロア・ターン数・Gold・クリアボーナスを難易度倍率で換算。ベストスコアはブラウザに保存されます

---

## 操作方法

1. タイルをタップして選択（最大3枚）
2. 「詠唱」ボタンで呪文を発動
3. 敵を倒してフロアを進む
4. ボス（魔竜）を撃破するとクリア
